# Backend

[1] OpenJDK 21.0.6 LTS 
LINK : https://learn.microsoft.com/en-us/java/openjdk/download#openjdk-21

[2] Maven 3.9.9
LINK : https://maven.apache.org/download.cgi

## Dokcer 啟動
### Windwos 10/11 用戶 Powershell

1. 切換到後端目錄
$ cd /backend

2. 編譯專案
$ . .\build.bat

3. 啟動專案
$ . .\up.bat

4. 關閉專案
$ . .\down.bat

### Linux

1. 切換到後端目錄
$ cd /backend

2. 編譯專案
$ sh build.sh

3. 啟動專案
$ sh up.sh

4. 關閉專案
$ sh down.sh
